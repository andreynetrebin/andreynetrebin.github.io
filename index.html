Обо мне

Моя цель занять позицию дата-инженера, чтобы применять свои навыки в обработке и анализе данных для создания эффективных и масштабируемых решений.

У меня есть опыт работы в различных областях обработки и анализа данных. Мои ключевые навыки включают:

  • Языки программирования: Python (скрипты, десктопные приложения с PyQT, веб-приложения с Flask и Django).
  • Обработка данных: Использование Pandas и PySpark для работы с данными из различных источников (базы данных, API, JSON, CSV, XML).
  • Визуализация данных: Создание интерактивных дашбордов с использованием Dash.
  • Автоматизация задач: Использование Selenium для автоматизации взаимодействия с веб-приложениями.
  • SQL и NoSQL базы данных: Опыт работы с DB2, Netezza, PostgreSQL, MySQL, ClickHouse, MongoDB
  • Инструменты и технологии: Опыт работы с брокерами очередей (IBM MQ, Kafka), стеком ELK, Apache Airflow, Docker и Camunda.
