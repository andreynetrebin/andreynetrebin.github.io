Обо мне

Моя цель занять позицию в области инжиниринга данных, чтобы применять свои навыки в обработке и анализе данных для создания эффективных и масштабируемых решений.

У меня есть опыт работы в различных областях обработки и анализа данных. Мои ключевые навыки включают:

  • Языки программирования: Python (скрипты, десктопные приложения с PyQT, веб-приложения с Flask и Django).
  • Работа с базами данных: Опыт работы с различными СУБД для аналитической обработки данных и выполнения сложных SQL-запросов, в том числе с MPP-системами IBM Netezza, Greenplum, ClickHouse.
  • Обработка данных: Использование Pandas и PySpark для работы с данными из различных источников (базы данных, API, JSON, CSV, XML).
  • Работа с HDFS: Опыт работы с Hadoop Distributed File System для хранения и обработки больших объемов данных.
  • Визуализация данных: Создание интерактивных дашбордов с использованием Dash и Grafana.
  • Автоматизация задач: Использование Selenium для автоматизации взаимодействия с веб-приложениями.
  • Инструменты и технологии: Опыт работы с брокерами сообщений (IBM MQ, Kafka), стеком ELK, Apache Airflow, Docker и Camunda.
  • Системы контроля версий: Уверенное использование Git для управления версиями кода, совместной работы над проектами и обеспечения надежности разработки.
